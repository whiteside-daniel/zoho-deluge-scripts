Bulk Upload - Instructions

File Limitations

Catalyst server can only handle 50MB of workload at one time. For groups of files larger than 50MB, use smaller batches. Each individual file is limited to 1MB.

File Format

File format must be zip files

You can upload multiple zip files at once

Each zip file must contain exactly one Line Item file and one Pacing plan file - per Bravo specification
