Features/Limitations

  • Catalyst only allocates 256MB RAM for serverless functions. Realistically this means a max of ~100MB of upload capability
  • Zoho People API limitation is 400 file upload calls in 5 minutes. So you should only upload 400 pdfs at once, and then wait 5 minutes or you will get errors.
  • You can select multiple files for upload in one dialogue, and the server will automatically sort and upload to Zoho People
  • Authentication can be limited to a single user, or email domain like user.email.Contains("@zoho.com"). Authentication is secured by Catalyst authentication service and provides equivalent security to the rest of Zoho applications.
  • If the server doesn't fail catastrophically, you will get a log file which contains success/fail data for each file.
